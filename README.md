# SQL Database Labs

**Author:** Somaia Mahmoud Shapaan

SQL Server labs covering JOINs, aggregation, subqueries, DDL with constraints,
CTE, window functions, schemas, CASE expressions, and table variables.

## Structure

```
sql-database-labs/
├── lab3_4/
│   └── queries.sql   # INNER/LEFT JOINs · GROUP BY · Aggregation · Subqueries · DDL
└── lab5/
    └── queries.sql   # CTE · DENSE_RANK · ROW_NUMBER · NTILE · Schema · CASE · Variables
```

## Topics Covered (Lab 3 & 4)
- INNER JOIN, LEFT JOIN, multi-table JOINs
- WHERE with OR, BETWEEN, LIKE
- Aggregation: AVG, MAX, MIN, SUM, COUNT with GROUP BY / HAVING
- Subqueries in WHERE and HAVING
- DDL: CREATE TABLE with constraints (PK, FK, CHECK, DEFAULT, IDENTITY)

## Topics Covered (Lab 5)
- Common Table Expressions (CTE)
- Ranking functions: DENSE_RANK, ROW_NUMBER, NTILE with PARTITION BY
- Schema creation and table transfer
- CASE / IIF expressions
- Table variables (DECLARE @var TABLE)
- SELECT INTO (with and without data)
